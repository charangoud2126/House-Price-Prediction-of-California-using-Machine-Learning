The main goal of this project is to build a predictive model that can accurately estimate house prices and help:

Real estate buyers make informed decisions

Sellers determine competitive pricing

Businesses perform property valuation analytics

## Dataset Features

The dataset includes important housing attributes such as:

Area / Size of the property

Number of bedrooms & bathrooms

Location-based factors

Property age and additional amenities

Target variable: House Price
